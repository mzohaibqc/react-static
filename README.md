# react-static-website

A barebones example of how to generate a static website with React, React Router, and Webpack.

## Getting Started

```
npm install
npm start
```

Open up [http://localhost:8080](http://localhost:8080) in your browser.

## Build

```
npm run build
```

The `dist` folder will contain the output of the build.

Directory structure:

```
project
│   README.md
│   package.json 
|   firebase.json
|   .firebaserc
|   webpack.config.json
|   .gitignore
|   LICENCE 
│
|___src
|   │   |   components
|   |   |   |   Layout.js
|   |   |   |   About.js
|   |   |   |   Contacts.js
|   
|___dist
|   |   index.html
|   |   bundle.js
|   |   style.css
```
